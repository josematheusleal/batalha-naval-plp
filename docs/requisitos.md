# Requisitos do Projeto – Batalha Naval

Este documento descreve os requisitos do projeto acadêmico Batalha Naval, desenvolvido para a disciplina PLP.

## Requisitos Funcionais

- O sistema deve permitir a criação de um tabuleiro de jogo.
- O tabuleiro deve permitir a consulta do estado de suas posições.
- O jogador deve poder realizar jogadas (ataques).
- O sistema deve controlar o andamento do jogo.

## Requisitos Não Funcionais

- O projeto deve ser desenvolvido em JavaScript.
- O código deve ser organizado e legível.
- O projeto deve utilizar GitHub para versionamento.
- O desenvolvimento deve seguir boas práticas de trabalho em equipe.

## Restrições

- O projeto deve seguir as regras definidas pelo professor.
- O prazo final de entrega é 02/03.
